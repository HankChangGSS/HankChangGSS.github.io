  1. step by step 實作

  • 在 Bitbucket 進入 Personal settings 頁面

  • 進入 App passwords 頁面

  • 填寫 Label ，勾選適當權限，按下 Create 按鈕

  • 看到彈現出的視窗後，記得儲存該密碼

  • 回到 Sourcetree 程式，點擊右上角的 Settings 齒輪圖示

  • 點選 Path 後，再按下 Edit 按鈕

  • 依下列規則調整 URL/Path

    • 假設先前取得的密碼是 1rTKwWrlMwyUesWe5jB3，在密碼前碼加上: 再放回原始設定值中的 @ 前方
  • 設定完成後，即可成功 push
